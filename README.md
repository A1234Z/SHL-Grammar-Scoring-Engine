# SHL Grammar Scoring Engine

This repository contains the notebook implementing a grammar scoring regression model for the SHL Audio Scoring Challenge on Kaggle.

## Methodology

- Extracted audio features (MFCC) using Librosa
- Built regression model (RandomForestRegressor)
- Evaluated via RMSE and Pearson correlation
- Generated submission.csv for Kaggle leaderboard

You can run the notebook from top to bottom to reproduce results.
